Intro

The map() method is used for arrays.

It allows us to easily to create a copy of the original array.

Can also make a modified copy of the original array.

The Call

Examples

A simple example of the map.

								
									const array = [1, 2, 3];

									let newArray = array.map();
									let modArray = array.map(e => e + 10);
Possible Uses

One: Use is to copy arrays. So you don't modified the original array.

Two: To copy and modify the copied array.
